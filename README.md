# Lumin OS
Welcome to the Lumin OS Main GitHub repository!

## What is Lumin OS?
Lumin OS is a OS made from the Windows Command Prompt (batch scripting). It includes a simulated computer, a simulated bootloader, and of course, a simulated OS (Lumin OS).
